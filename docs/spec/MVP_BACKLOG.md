# MVP Backlog

## Phase 0 (Weeks 0-2)
- Define taxonomy v0 for AI landscape.
- Establish memo template and scoring rubric.
- Identify core data sources and licensing.

## Phase 1 (Weeks 3-6)
- Ingestion for filings, news, macro releases, and market data.
- Entity resolution and normalized storage.
- Evidence pack generation with provenance.
- Daily AI landscape brief generation.

## Phase 2 (Weeks 7-10)
- Memo authoring pipeline with versioning.
- Analyst swarm scoring and consensus aggregator.
- Theme and company pages with evidence views.

## Phase 3 (Weeks 11-14)
- Macro transmission map and causal graph views.
- Alerts and watchlists.
- Outcome tracking for memo accuracy.

## Phase 4 (Weeks 15-20)
- Counter-thesis agents and red-team reviews.
- Evaluation dashboard and calibration tooling.
- Cost controls and caching improvements.
